# Sun Bowen Portfolio

高级暗黑风个人求职主页，定位为软件工程背景、PMP 项目管理思维、产品与项目协作能力结合的个人作品站。

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- GSAP ScrollTrigger
- Three.js lightweight particle background
- Vitest

## Run

```bash
npm.cmd install
npm.cmd run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Content

Personal copy and structured content live in `src/data/profile.ts`.

Replace `public/resume.pdf` with the final resume PDF before sending applications.

## Verify

```bash
npm.cmd test -- tests/profile.test.ts
npm.cmd run lint
npm.cmd run build
```
